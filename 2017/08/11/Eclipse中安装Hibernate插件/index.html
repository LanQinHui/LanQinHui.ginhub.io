Eclipse中安装Hibernate插件

发表于 2017-08-11 | | 浏览 次
字数统计 196 字 | 阅读时长 1 分钟

在使用hibernate开发时，大多数情况下涉及到其XML配置文件的编辑，尤其是.cfg.xml（配置文件）和hbm.xml（关系映射文件）这两种。为了更方便的使用此框架，其插件的安装是很有必要的。

一、下载Hibernate框架的插件：JBOOS Tool

官网链接： http://tools.jboss.org/downloads/overview.html



二 安装插件

打开Eclipse—>Help—->Install New SoftWare—–>add—->Archive（找到你的插件–打开）——>Ok——>选中 JBoss Tools hibernatetools Nightly Build Update Site

—–>next—–等待一段时间—->next—–>next—–>I accept…..—->finish——>开始安装Install SoftWare——>安装好之后会问你是否重启Eclipse，点击OK（需要restart eclipse）


关联提示 路径：\hibernate-release-5.2.2.Final\project\hibernate-core\src\main\resources\org\hibernate之后就有提示了。

之后就会有提示了
